# 🌍 Global Travel Guide

AI-powered travel planner built with Streamlit that provides personalized recommendations for any city worldwide.

## 🚀 Features

* 🌆 Works for any global city
* 🏨 Top hotel recommendations
* 🍽 Best restaurants & cuisines
* 📍 Must-visit attractions
* 🚗 Budget-based travel suggestions
* 🤖 AI-generated travel plans (Gemini)

## 🛠️ Tech Stack

* Python, Streamlit
* Pandas, Requests
* Google Generative AI (Gemini)
* OpenStreetMap APIs

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run GlobalTravelPlanner.py
```

## 🔐 Setup API Key

Create `.streamlit/secrets.toml`:

```toml
GOOGLE_API_KEY = "your_api_key_here"
```

## 📌 Use Case

Helps travelers quickly plan trips with data-driven and AI-powered insights.

Interactive dashboard with recommendations, travel tips, and AI itinerary generation.
